# Объекты Canvas

## О проекте

Интерактивное веб-приложение для **визуализации 2D объектов и связей** в рамках HTML-элемента canvas.

## Архитектура приложения

Компоненты, функции, хуки и типы приложения разбиты на 3 директории: **components, lib, types**.

### components:

Содержит визуальные компоненты приложения: коревой `App` и компонент для работы с графом `GraphEditor`.

### lib:

Содержит вспомогательные функции, например, **рисование объектов/связей** или **поиск объекта по координатам на канвасе**.

Внутренняя директория hooks содержит хук `useCanvas` для **синхронизации размера контейнера с размером канваса** и `useGraph` для **работы с данными** (добавление/удаление).

### types

Хранит типы данных для сущности объекта и связи.

## Быстрый старт

### Предварительные требования

Необходимая версия Node.js 24.12.0

### Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/teddehhh/canvas-objects
   ```
2. Перейдите в директорию проекта
   ```bash
   cd canvas-objects
   ```
3. Установите зависимости
   ```bash
   npm i
    # или
   yarn i
   ```

### Запуск

1. Запустите сервер
   ```bash
   npm run dev
   # или
   yarn run dev
   ```
2. Откройте http://localhost:8080 в браузере.
